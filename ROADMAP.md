# Scheme Mate — Project Roadmap

This roadmap tracks the development progress of **Scheme Mate — Find. Apply. Benefit. (Your Smart Government Benefits Assistant)**.

---

## 🗺️ Module Timeline & Status

*   **✅ Module 1: Authentication**
    *   JWT Authentication, Password hashing, Express API, Flutter validation.
*   **✅ Module 2: User Profile**
    *   Socio-economic eligibility parameters, cascading State-District-Taluk selections, JSONB documents checklist, profile completion score.
*   **✅ Module 3: Scheme Database**
    *   Curated schemes table (UUID keys), GIN index full-text search, transactional scheme version history tracking, Swagger API docs.
*   **✅ Module 4: Rule-Based Eligibility Engine**
    *   Demographic, geographic, and document rule evaluation, status resolution (`Eligible`, `Partially Eligible`, `Not Eligible`, `Unknown`), and detailed mismatch reasons logs.
*   **✅ Module 5: Personalized Dashboard**
    *   Overview statistics, matched schemes highlight deck, trending schemes, and documents verification alerts.
*   **✅ Module 6: Saved Schemes**
    *   Saves counters management, user bookmark list endpoints, notes updates, offline caching sync, and deadline reminders.
*   **✅ Module 7: Notifications**
    *   Alerts on scheme updates, document expirations, new schemes matching, prioritizations, and settings preferences.
*   **✅ Module 8: Feedback & Bug Report**
    *   Simple submit endpoint for bug reporting, incorrect scheme alerts, structured error logging redirections, and popup dialog overlays.
*   **✅ Module 9: Version 1.0 Beta Stabilization & QA Testing**
    *   E2E manual verification checklists, logging monitoring, offline testing, security validations, and query audits.
*   **⏳ Module 10: Closed Beta & User Testing** *(Current)*
    *   Deploying to real users (friends, family, farmers, students) to collect bug and UX feedback.
*   **⏳ Module 11: Document Vault**
    *   Local secure storage configuration, file upload options, and metadata tagging.
*   **⏳ Module 12: OCR Document Scanner**
    *   Mobile optical character reading to auto-extract details (such as names, dates, categories) and auto-fill profile forms.
*   **⏳ Module 13: Multilingual Support**
    *   Localization packages translating forms and scheme databases into local Indian languages (Kannada, Hindi, etc.).
*   **⏳ Module 14: AI Assistant**
    *   Natural language chat advisor answering eligibility doubts, guiding through application forms, and querying the schemes database.
